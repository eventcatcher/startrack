ROPOCOP

Die ROmPOrt-COntroller-Platine für Hades

                                                                          updated 12.7.1997

ROPOCOP ist die neue Romport-Karte für den Hades. Sie bietet 3 softwaremäßig
(neues Hades-Accessory) umschaltbare Romports (1 x extern, 2 x intern).
Die Karte wird einfach in einen der beiden ISA-Slots eingesteckt - fertig.

Die Karte wurde hauptsächlich dafür entwickelt, damit die meist durch Romport-Dongles
geschützte Musik-Software auch auf dem Hades läuft.

Der Preis wird vorraussichtlich um die 200 DM liegen.



Folgende Dongle-Software wurde bereits mit Erfolg getestet und läuft auf dem Hades:


Soundpool CD-Recorder 2 - Hiermit kann man eigene CDs brennen (Daten-CDs,Musik-CDs usw.)

Emagic Logic 2.5 - Ein sehr leistungsfähiger MIDI-Sequenzer

Steinberg Cubase Audio Falcon 2.6 - Ebenfalls ein sehr leistungsfähiger MIDI-Sequenzer,
läuft ohne Falcon-Audioteil (vergleichbar mit Cubase Score)


Alle oben aufgeführten Programme laufen auf der 040er und jetzt auch auf der 060er CPU !
