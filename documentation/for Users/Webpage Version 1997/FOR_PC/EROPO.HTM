ROPOCOP

The ROmPOrt-COntroller-PCB for Hades

                                                                          updated 07/12/1997

ROPOCOP ist the new Romport-Card for the Hades. It extends your Hades with
3 software-switchable (new Hades-Accessory) Romports (1 x external, 2 x internal).
The card you easy plugged into one of the both ISA-Slots - ready.

The card is especially developed for using dongle-protected Music-Software with the Hades.

The price is probable ca. 200 DM (1US$ is ca. 1.70 DM)



This dongle-protected-Software is tested with it and runs successful on the Hades:


Soundpool CD-Recorder 2 - for burning your own CDs with a CD-Writer (Data-CDs,Music-CDs etc.)

Emagic Logic 2.5 - a very powerful MIDI-Sequencer

and....Steinberg Cubase Audio Falcon 2.6 - the very powerful MIDI-Sequencer
runs without Falcon-Audio (comparable with Cubase Score)


All this programs run now with a 040 and - 060 CPU !
