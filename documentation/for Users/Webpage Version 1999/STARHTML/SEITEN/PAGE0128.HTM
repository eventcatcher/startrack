 5.1.17 Cuelist
ALT B Export zum CD-Brenner (via Extendos Gold)
ALT M Downmix der Cuelist
ALT K Kill Cuelist (löscht alle Einträge)
Pfeil runter Eintrag auswählen (+1)
Pfeil hoch Eintrag auswählen (-1)
LSHIFT Pfeil runter Eintrag auswählen (+5)
LSHIFT Pfeil hoch Eintrag auswählen (-5)
RSHIFT Pfeil runter Eintrag auswählen (+10)
RSHIFT Pfeil hoch Eintrag auswählen (-10)
CTRL L Load Cuelist
CTRL S Save Cuelist
CTRL Pfeil runter Cuelist-Darstellungsbereich höher (+1)
CTRL Pfeil hoch Cuelist-Darstellungsbereich niedriger (-1)
CTRL LSHIFT Pfeil runter Cuelist-Darstellungsbereich höher (+5)
CTRL LSHIFT Pfeil hoch Cuelist-Darstellungsbereich niedriger (-5)
CTRL RSHIFT Pfeil runter Cuelist-Darstellungsbereich höher (+10)
CTRL RSHIFT Pfeil hoch Cuelist-Darstellungsbereich niedriger (-10)
