 5.2.5 Musicbox-Datei "*.BOX"

In diesen Dateien werden die Musicbox-Einstellungen des Audio-Fileselectors
abgespeichert. Da der Dateiname frei wählbar ist, können mehrere dieser
Musicbox-Dateien angelegt werden. So kann man individuelle Konfigurationen
für verschiedene Verwendungszwecke zusammenstellen.
entrys Anzahl der Musicbox-Einträge
namexyz Dateiname des Samples von Eintrag-Nummer xyz
pathxyz Kompletter Dateipfad mit Dateiname von Eintrag-Nummer xyz
